# Sentiment Analysis Project

A Machine Learning based Sentiment Analysis system that predicts whether a text/review is Positive or Negative using NLP techniques.

## Features

* Text preprocessing using NLTK
* TF-IDF Vectorization
* Sentiment Prediction
* Saved trained models using Pickle

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* NLTK

## Run Project

```bash
pip install -r requirements.txt
python3 sentimental_analysis.py
```
