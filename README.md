# Customer Sentiment Analysis & Topic Modeling

A Data Analytics and Natural Language Processing project focused on analyzing customer sentiments from textual data using Python.

## Project Overview
This project analyzes customer reviews and textual feedback to identify sentiment patterns and extract key discussion topics. The project demonstrates practical data analysis, text preprocessing, visualization, and predictive analytics skills using NLP techniques.

## Key Features
- Text data preprocessing and cleaning
- Sentiment classification using Logistic Regression
- TF-IDF feature engineering
- Topic Modeling using Latent Dirichlet Allocation (LDA)
- Model evaluation using accuracy metrics and confusion matrix
- Saving trained models for future predictions

## Technologies Used
- Python
- Pandas
- NumPy
- NLTK
- Scikit-Learn
- Joblib

## Dataset
The dataset contains:
- Customer reviews / text data
- Sentiment labels
- Preprocessed textual features

## Project Workflow
1. Data Collection
2. Text Cleaning & Preprocessing

5. Topic Modeling (LDA)

7. Model Saving & Prediction
